# Sistema de Chamados em Java

Projeto simples que simula um sistema de abertura de chamados de suporte técnico.

## Funcionalidades

- Abrir chamado
- Registrar usuário
- Registrar problema
- Listar chamados

## Tecnologias

- Java
- Programação orientada a objetos

## Como executar

1. Compile o programa
2. Execute a classe SistemaChamados
3. Utilize o menu no terminal
